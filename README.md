# RPG Adventure (Matricola 125556)
Questo progetto consiste in un gioco di ruolo (RPG) a turni sviluppato in Java e JavaFX. Il gioco si concentra sul sistema di combattimento e implementa sistemi di:
- level up e allocazione punti statistica
- inventario
- equipaggiamento
- salvataggi persistenti in formato JSON
il tutto con un'interfaccia utente grafica.

## Come eseguire il progetto
### Prerequisiti:
- Java 25 (LTS)
- Gradle

### Istruzioni:
  cloning:
  ```bash
  git clone https://github.com/Davide-Dalessio/MdP
  cd MdP
  ```
  Build:
  ```bash
  ./gradlew build
  ```
  
  Esecuzione:
  ```bash
  ./gradlew run
  ```
